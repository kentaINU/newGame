# 🚀 プロジェクト実行ガイド

このプロジェクトは Docker (Dev Containers) を使用して、React + Vite の開発環境を構築します。

## 📋 事前準備
1. **Docker Desktop** が起動していることを確認してください。
2. **VS Code** に拡張機能 **「Dev Containers」** がインストールされていることを確認してください。

---

## 🛠 セットアップと起動手順

### 1. コンテナのビルドと接続
1. VS Code でこのプロジェクトのフォルダを開きます。
2. 画面左下の **青い [><] アイコン** をクリックします。
3. 表示されたメニューから **「Dev Containers: Reopen in Container」** を選択します。
4. コンテナの作成が完了するまで待ちます（初回は数分かかります）。

### 2. 起動方法
`frontend` フォルダに移動後、以下のコマンドでゲームを起動します：

```bash
npm run dev
